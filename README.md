## Quick Start

Rename .env.local 
```
/api/.env.local > /api.env
```

**Run react dev server**

```bash
yarn start
```

**Run node.js express server.**
```bash
cd api/
node server.js
```

**Nodemon**
```bash
cd api/
nodemon server.js
```
Visit http://localhost:3000

Backend located at http://localhost:3001
